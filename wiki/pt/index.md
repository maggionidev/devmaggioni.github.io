# 📚 Maggioni.dev WIKI

![maggioni.dev](https://assets.maggioni.dev/images/maggioni.dev.png)

Documentação oficial, decisões técnicas e guias para o ecossistema e desenvolvimento do **maggioni.dev**. Este projeto é construído utilizando o framework **Hugo** com o tema **PaperMod**.

---

## ⚡ Stack & Arquitetura

* **Gerador de Site Estático (SSG):** Hugo
* **Tema Base:** PaperMod
* **Hospedagem:** GitHub Pages
* **Domínio e DNS:** GoDaddy (gerenciado com Cloudflare)
* **Armazenamento de Assets:** Backblaze B2 (proxy via Cloudflare Workers)
* **Princípios Fundamentais:** Performance, simplicidade, privacidade e controle de dados.

---

## 🧠 Por que Hugo?

O Hugo foi escolhido devido à sua velocidade, estabilidade e formato nativo em Markdown. Os principais diferenciais para este projeto são:

* **Performance:** Geração de builds quase instantânea.
* **Leveza:** Ausência de dependências de *runtime* pesadas (como Node.js) em produção.
* **SEO e Acessibilidade:** Foco em código semântico e rápido.
* **Propriedade e Controle:** Facilidade de versionamento e portabilidade do conteúdo.
